### Приложения для заметок (Тестовое задание ШИФТ Лаборатория)
## Постановка задачи:
Необходимо реализовать приложение для создания заметок. 

Созданное приложение имеет следующую функиональность:
1. Создание новой заметки (FAB на основном экране, чтобы сохранить заметку необходимо нажать иконку "Готово")
2. Просмотр списка всех созданных заметок
3. Удаление заметок (свайп влево)
4. Просмотр созданной заметки (нажатием на заметку)
5. Редактирование существующей заметки (нажатием на заметку)
6. Возможность отметить заметку тэгом "Избранное" (нажать на иконку "Избранное")
7. Поиск заметок по заголовку и тексту (Вввод на экране всех заметок, можно очистить ввод, нажав на соответсвующую иконку)
8. Сохранение черновика новой заметки (если пользователь вышел с экрана добавления заметки кнопкой Назад, а не нажал на кнопку Сохранить, то при следующей попытки создать новую заметку подставится текст из черновика)
9. Поиск заметок по тэгу "Избранное" (Вкл/выкл иконка "Избранное" на экране всех заметок)
10. Просмотр информации о заметки (дата последнего изменения, кол-во символов в заметке, дата напоминания)
11. Установка и удаление напоминания на заметку (нажатием на иконку "Уведомление" только у уже существующей заметки)

--------------
## Стек:
min SDK = 31, Kotlin, Clean Architecture, multi-module, MVI + JetPack Compose, DI - Koin, Room Database, Jetpack Navigation
